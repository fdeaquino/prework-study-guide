HTML

  • HTML stands for Hypertext Markup Language and provides the STRUCTURE of a webpage
  • Think of HTML as the frame of a house.
  • The head element contains information about the webpage.
  • Elements included within the head are not visible when a user looks at a webpage.
  • The title of a webpage is pretty important for Search Engine Optimization. SEO is the process of improving the ranking of a website in search engines like Google or Bing.
  • The body element represents the visible content shown to the user.
  • All elements that we want the user to see and interact with must be placed inside the body element.
  • Within the body section there are a lot of HTML elements. Look at notes on iPad for the break down. :)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git fetch: downloads all history from the remote tracking branches
  • git merge: combines remote tracking branch into current local branch
  • git push: uploads all local branch commits to GitHub
  • git pull: updates your current local working branch with all new commits from the corresponding remote branch on Github
  • git pull: is a combination of git fetch and git merge
